## Run Locally

```bash
# Install dependencies
npm install

# Set API key in .env.local
VITE_GEMINI_API_KEY=your_key_here

# Start dev server
npm run dev
```

## Deployment

```bash
# Build for production
npm run build

# Deploy to Firebase
firebase deploy --only hosting
```
